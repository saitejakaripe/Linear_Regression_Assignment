# Linear Regression Assignment

A linear regression project focused on analyzing and predicting bike-sharing demand using exploratory data analysis, preprocessing, and regression modeling.

## Overview

This project applies linear regression to understand the factors affecting bike demand and build a predictive model using historical data.

## Objectives

- Explore the dataset
- Identify important demand drivers
- Build a linear regression model
- Interpret model coefficients
- Evaluate model performance

## Tech Stack

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
